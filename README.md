# Online card game: Higher takes it all
Online card game for two players. The client joins a lobby, is matched with another player, and the game begins.

Application implements: multithreading, client reconnection, its own communication protocol, ping messages.
## Client
 - has GUI
 - written in Java
 - runs on Linux and Windows
## Server
 - written in C
 - runs on Linux

More about the project and its functions can be found in Czech-written [documentation](Dokumentace.pdf).
